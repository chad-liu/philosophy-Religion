## 網站結構

### 首頁（index.html）

提供四個跨講整合的索引 tab：

- **課程**：15 講章節清單與連結
- **專有名詞**：跨 15 講整合，依主題分組，共 116 條術語
- **時間軸**：去重合併為 5 個時代分期，共 78 條歷史記錄
- **重要人物**：依西方人物／東方人物分類，共 84 位

### 各講頁面（R01.html – R15.html）

每講包含 6 個 tab：

| Tab | 內容 |
|-----|------|
| 完整稿 | 逐字整理的全文講稿 |
| 大綱 | 章節大綱與重點摘要 |
| 專有名詞 | 該講術語與定義 |
| 時間軸 | 該講涉及的歷史年表 |
| 重要人物 | 該講出現的人物介紹 |
| 來源影片 | 連結至 YouTube 原始課程影片 |
