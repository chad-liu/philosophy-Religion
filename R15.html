宗教哲學總結：信仰、生死與人生智慧

宗教哲學的核心界定

我們今天來做個總結。前面談過很多有關什麼是宗教，宗教哲學就是對宗教做哲學的反省，而要做哲學反省，首先一定要清楚界定什麼是宗教，因為宗教的現象太複雜了，在社會上有各種面貌，連信徒自己有時也分不清楚哪些才是真正的宗教核心。

我在課堂上一再強調，宗教的核心是信仰，信仰是人與超越界之間的關係，這是基本的界定。聽起來好像很簡單，但這是我研究這門學科數十年的心得。有關宗教哲學的界定，隨便都可以舉出十種八種，但我的這種理解，是最核心最重要的部分。哲學跟宗教方向是一樣的，都是要去探討最真實、最究竟的真理，但方法不一樣。宗教的方法是信仰，信了就不需要再懷疑，後面是實踐的問題；哲學的方法是理性，是開放的，可以一直不斷懷疑、不斷討論，隨著個人生命經驗的成長而顯示多彩多姿的樣貌。

超越界的兩種形態

確立了超越界的概念之後，接著的問題是：超越界是什麼形態的存在？這就分出了兩個大系統。第一個系統，是把超越界當作有位格的，就形成了三大一神教：猶太教、基督宗教、伊斯蘭教。這個系統在西方哲學史上發展出一個非常重要的主題，叫做上帝存在論證。

這個論證你就算證明了，也只是哲學家的上帝，跟宗教裡面聖經的上帝不見得是同一個概念。但是你如果不了解這個論證，不知道為什麼西方人在討論哲學的時候要不斷碰到這個問題，你就不可能了解哲學所說的「愛好智慧」，最後還是要回到最根源的問題。

第二個系統，是把超越界當作非位格的，它只是一種境界，像涅槃，或是梵我一如這樣的狀態。這個系統的代表當然是印度教跟佛教。每個宗教到了修行的最高層次，都希望達到一種密契的經驗，就是人跟他信仰的對象合而為一，形成一個原始的整體。這也暗示了人生困難的根源：就是有個自我意識。沒有自我哪裡有人生？但是有了自我之後，一生都在尋求解脫，密契經驗就是自我的解脫。

輪迴的好處與省思

三大一神教主張審判，印度教與佛教主張輪迴，這是對死後正義問題的兩種不同回答。輪迴的好處是什麼？它可以緩和人對公平的要求。人生充滿了不公平：出生不公平，有些人生下來就有身心殘障、貧困、缺乏親情；人生的遭遇不公平，遇到戰爭或是不公平的法官；死後你還要求這一生的善惡有所報應，這本身也是一種要求公平的心理。輪迴的好處就是可以把這個公平延伸到前世跟來世，讓你比較願意接受現實的情況，知道好人今生受苦是前世的業，壞人今生得意是來世要還的。

但輪迴有幾個值得思考的地方。第一，輪迴的範圍是什麼？如果只限於人類，人類何以越來越多？靈魂可以分化嗎？如果包括眾生在內，問題又來了：別的動物沒有清楚的自我意識跟自由意志，牠們的輪迴要以什麼為準？因為要有善惡之分，一定先要有自我跟自由。有本書記載，一位基督教牧師問印度教 Guru：「蚊子怎麼輪迴成人？」那 Guru 說：「蚊子去叮一個壞人，被打死，下輩子就輪迴成人了。」所以我們都是運氣好的蚊子——這話說出來令人哭笑不得，但你講眾生輪迴，確實很難避免這樣的問題。

第二，輪迴的規則是什麼？做多少壞事變成蚊子，做多少壞事變成豬，這個規則誰來定？第三，輪迴的目的何在？如果說最後都可以解脫，那何必負責任呢？如果終究不能解脫，那還是需要某種形式的審判——輪迴不能無限制地延伸下去。

審判的問題

一神教主張一次性的審判，沒有重來、沒有上訴。這強調了今生今世的個人價值與尊嚴，讓人知道這一生的選擇是真實有分量的，可以悔改，但也需要自負其責。

但審判也有困境。第一，無法解釋出生的不公平。如果我生下來就在不公平的起跑點，最後的審判要如何公平評估？就像橋牌比賽如果不換牌，一直拿到好牌的人天生就有優勢。第二，審判預設人能夠正確辨別善惡，但這一點並不容易做到。從小在惡劣環境成長的人，他的善惡觀念跟我們可能完全不同。第三，審判預設人有自由意志，但我最近感冒吃藥之後才深刻體會到人沒有自由——藥吃下去之後我要念書，完全不行，昏倒了，藥物太厲害，完全控制了我的身體。這讓我比較能體諒那些吸毒酗酒之後去犯罪的人。

我常常有一種自覺：如果我是他，我可能比他還要更糟糕；如果他是我，他可能做得比我更好。這兩句話一想，對每一個人都會有一種體諒的心。耶穌說得好：「不要只看到別人眼中的小草，而看不到自己眼中有一根大樑。」審判是上帝的事，我們人類彼此的善惡差別是相對的，我們有什麼必要每天去審判別人呢？

天堂、地獄與煉獄

天主教在天堂與地獄之間，還有一個廣大的煉獄，讓靈魂繼續淨化。煉獄裡的人終究都會升天堂，活著的親友也可以替他們祈禱，在佛教叫做回向，意思是一樣的。不過有個問題：你說「終究」升天堂，這個「終究」是多久？很久很久之後。但時間只對有身體的人有意義，對沒有身體的靈魂，一千年兩千年又有什麼感覺呢？

關於天堂和地獄，有個比喻我覺得非常生動。天堂和地獄裡的人都需要吃東西，但是每個人只有一種很長的筷子。天堂的人從小養成幫助別人的習慣，拿食物喂別人，別人也喂他，所以各個紅光滿面。地獄的人從來不願意幫助別人，筷子太長自己夾不到口裡，又死都不肯喂別人吃，因為他不相信別人會回報他，所以在地獄裡餓得要命。這只是個比喻，不是說地獄的人學會了互相照顧就能升天堂，但這個比喻說的就是一個人的根本態度決定了他的世界。

比較合理的人生態度

我們活在世界上，要面對四個領域：自我、群體、自然界、超越界。這四個分類是完整的，不能有遺漏。對自我，要覺悟自我的結局——人生注定要面對痛苦、罪惡和死亡，由此才能與超越界建立真正的關係；對群體，要關懷，但不受其控制，因為群體可能腐化；對自然界，要欣賞，但不要沉迷其中；對超越界，建立信仰關係。

人的生命也有三個層次：身體、心、靈。身體是有形的，互相排斥，空間是有限的；心智可以溝通，知識可以分享；靈性是屬於全人類的，與超越界相通，一旦提升到靈的層次，人與人之間不再有根本的衝突，打成一片。

還有一個重要的課題：孤獨。人孤獨地來，孤獨地走。製造許多活動讓自己暫時避開孤獨，但孤獨的根本問題沒有解決。我們要學習從忍受孤獨，到接受孤獨，再到享受孤獨——在孤獨裡面跟超越界建立關係，這就是靈性轉向最重要的起點。

儒家作為橋樑

儒家的人性向善論，在各宗教之間可以扮演橋樑的角色。我長期堅信這一點。為什麼？因為我所說的儒家，不是人性本善。如果說人性本善，就跟基督宗教的原罪說無法相容，也跟佛教的性空論無法相容，這也是宋明理學對佛教排斥不遺餘力的原因。

我說的儒家從孔子開始：人性向善，只要真誠，就有力量由內而發，主動要求自己去行善。真誠是儒家的核心，也是所有宗教共同的基礎。孔子說「道不同，不相為謀」，各走各的路，但走到最後還是可以互相欣賞、互相尊重。宗教之間的溝通能做到什麼程度？最多就是「我了解你，但不代表我贊成你」，雙方相互理解、彼此尊重，這就足夠了。

落實到日常生活，無緣大慈、同體大悲是很好的理想，但怎麼落實？不必去愛全世界——我坐在車上心裡想愛全世界，旁邊一個老人家站著我不讓座，愛全世界又有什麼用呢？從身邊正好出現的人開始就夠了。天下沒有真正的偉大，除非你能在身邊的小事上開始做起。

高級宗教與低級宗教

英國歷史學家湯恩比給了三個標準，我認為受他影響很大。高級宗教，第一個特徵是肯定人性不完美、人生不夠理想。所以你才需要跟超越界建立關係，往上提升。你一說人性本善、人生美好，就已經在說低級宗教了，因為你根本不需要超越。

第二，高級宗教必定反對並致力消除人間的罪惡，跟罪惡宣戰，絕不妥協。

第三，高級宗教坦然接受一切苦難，視之為磨練與解脫之途。佛教四聖諦第一個就是苦，正面面對它，不逃避，後面再問苦怎麼來的、怎麼化解。如果有一種宗教跟你說信了就可以過幸福的生活，那就是低級的。我記得以前有種宗教是號召信徒買股票，說信了我之後我帶你們買，一定賺錢——那是低級又低級。

迷信的四個特徵

迷信有四點特徵。第一，出於恐懼心理，無法引發積極的愛心。你信一種宗教是因為不信的話會下地獄，那是恐懼；你應該因為這個宗教啟發了你的愛心，信了之後更能慈悲博愛，這才是正信。

第二，崇拜特定的人，也就是教主，以至於無法轉向真正的超越界。佛教說得好：依法不依人，依的是佛所說的智慧與解脫之道，而不是某個人。當然耶穌是個例外，他宣稱自己是神，你要看他說什麼話、做什麼事，而不是看他有多少存款、坐什麼車。有人問我說兩千年後你怎麼知道宋七力不會變成第二個耶穌？我就回答一句話：耶穌有存款嗎？耶穌說過，狐狸有洞穴，飛鳥有鳥巢，我沒有放枕頭的地方。聖經裡哪一行可以讓你看出耶穌有任何財產？

第三，迎合世俗的欲望，使神明成為功能神、功利神。有的廟保佑感情，有的廟保佑生兒子，有的廟保佑買股票賺錢——每個廟都有不同的專長。

第四，執著於宗教的排他性，敵視欺壓教外之人。有什麼理由排斥別人呢？別人不信你這個教並不代表你對或他錯，信仰自由是憲法規定的基本原則。

宗教的現狀與未來

宗教多元化是自古以來就存在的事實，每個人的機緣不同。宗教信仰需要個人的機緣與感動：從小有什麼樣的家庭背景，就容易被某個宗教感動。我看電影鐵達尼號，快沉的時候，有幾個神父拉著信徒的手，一直念著「走過死亡的幽谷」，船員弦樂隊在那邊奏一個旋律奏到最後，那旋律只有天主教才有，意思就是「神啊，求你可憐我們」。就那麼簡單，每個宗教在生命臨終的時候，都有自己最後的那個願望：進入天堂，進入極樂世界，只希望能得到赦免和安慰。

宗教的最大困境是世俗化。耶穌說過：「你們是地上的鹽，鹽如果失去味道，什麼能使它再鹹呢？」再大的寺廟、再宏偉的教堂，如果信仰消失了，就只剩下建築。終極關懷漸趨模糊，這是當代宗教最大的威脅。

各宗教之間的溝通，在教義的層次是很難完全融合的。天主教相信神創造世界，佛教認為世界是因緣合和而生，你怎麼在這個層次溝通？只能說我了解你，但不代表我贊成你，雙方相互理解、尊重，就這樣。宗教內部的基本教義派，目的應該是保持信仰的純度，而不是淪為宗教間互相排斥的工具。

哲學的高低之分

哲學沒有對錯，只有高低。判斷標準是涵蓋的範圍跟解說的效率。我們在這個世界上見到四個層次：物質、生命、意識、精神。物質有惰性，可以被控制；生命有自我更新的能力；意識有直接意識，動物也有，知道冷不冷、有沒有吃的；精神是反省意識，就是意識到自己，也就是自我意識，只有人類才有。

唯物論把所有東西都用物質來解釋，當然可以講得通，但涵蓋面很窄，解釋效率受到限制，很難解釋意識跟精神的現象。能夠涵蓋四個層次的哲學，位階自然比較高。同樣道理，宗教的高低，就看它能不能維持並提升人與超越界之間的關係，這是核心。

信仰的最終意義

這個課程的結論，我可以說一句話：宗教信仰需要靠個人的機緣與感動，而這種信仰是與超越界建立關係。要記得，有關係一定會產生改變，如果建立了關係而沒有改變，那是假的關係。信仰永遠是一個抉擇，是一個冒險，是讓自己變得不同的力量。

存在就是製造差異。宗教信仰的對象是否存在，要看它能不能在你身上製造差異，讓你變成不同的人。真正做個人，應該是完整的，就是要有靈性的生命。宗教信仰應該是生命不可或缺的部分，是一切意義的基礎，由此才能使人真正在當下體驗一切在一切之中。

好，我們這個課就上到這裡。

宗教哲學總結：信仰、生死與人生智慧

一、核心命題回顧

  • 哲學與宗教：方向相同（探討最真實的真理），方法不同（哲學用理性，開放；宗教用信仰，信後實踐）
  • 宗教的核心定義：信仰是人與超越界之間的關係（超越界既非自然界也非人類）

二、宗教的兩大系統

| 系統 | 超越界性質 | 代表宗教 |

|------|-----------|---------|

| 第一系統 | 有位格（Personal God） | 三大一神教（猶太教、基督宗教、伊斯蘭教） |

| 第二系統 | 非位格（境界，如涅槃、梵我一如） | 印度教、佛教 |

上帝存在論證：即使成功，只能證明「哲學家的上帝」，未必等同聖經的上帝；但不理解此論證就無法理解西方哲學。

三、輪迴的四個省思

好處：緩解公平的張力

將公平延伸至前世與來世（甚至無限世），使現世不公平更易接受。

四個問題

| 問題 | 說明 |

|------|------|

| 範圍問題 | 若限人類，人口為何增加？靈魂能分化？若含眾生，動物無自我與自由，如何承擔善惡因果？ |

| 規則問題 | 做多少壞事變蚊子/豬/細菌？規則由誰定？會有誤差嗎？ |

| 目的問題 | 若輪迴最終都能解脫，何必負責任？若終究不能解脫，仍需某種審判 |

| 責任問題 | 可能推卸責任（前世做壞事，這輩子不關我事） |

四、審判的省思

優點：強調個人價值與尊嚴，鼓勵個人自負其責（一次性，不能上訴）

三個困境：

1. 無法解釋出生的不公平（起跑點不同）

2. 遭遇不公平（同樣犯罪，不同法官判決不同）

3. 需要人能正確辨別善惡、具備純粹的自由意志（均難保證）

悔改的問題：真誠懺悔是可能的，但無法保證不再犯。顏回「不遷怒、不二過」是孔子最高讚美的標準——普通人只能不斷犯錯、不斷懺悔。

煉獄：天主教的中介空間，供靈性淨化，最終升天堂；但時間對無肉體者是否有意義，值得思考。

關於審判的態度：耶穌說「不要只看別人眼中的小草，看不到自己眼中的大樑」——審判是上帝的事，人類善惡差別是相對的。

五、比較合理的人生態度

人生四大領域

| 領域 | 態度 |

|------|------|

| 自我 | 覺悟自我的結局（死亡、痛苦、罪惡），由此與超越界建立關係 |

| 群體 | 關懷群體，但不受其控制（群體可能腐化） |

| 自然界 | 欣賞，但不沉迷 |

| 超越界 | 建立信仰關係，靈性轉向 |

自我的三個層次

| 層次 | 特色 |

|------|------|

| 身體 | 有形，互相排斥（空間有限） |

| 心（知情意） | 可溝通，可分享 |

| 靈 | 屬於全人類，與超越界相通，打成一片 |

孤獨的三個層次

  • 忍受孤獨 → 接受孤獨 → 享受孤獨（在孤獨中與超越界建立關係）

六、儒家的橋樑角色

  • 人性向善論（非本善）：真誠則有力量由內而發，主動行善
  • 不說本善，故與基督宗教（原罪）、佛教（性空）均可相容
  • 宗教之間的溝通上限：「我瞭解你，不代表我贊成你」
  • 落實慈悲博愛：不必愛全世界，從身邊正好出現的人開始即可

七、高級宗教的三個標準（湯恩比）

| 標準 | 說明 |

|------|------|

| 肯定人性不完美、人生不理想 | 所以才需要信仰與超越，否則何必信？ |

| 反對並消除人間的罪惡 | 與罪惡宣戰，絕不妥協 |

| 坦然接受苦難，視之為磨練與解脫 | 不欺騙信徒說「信了就幸福」 |

低級宗教的反例：承諾信了就過幸福生活、號召信徒投資股票。

八、迷信的四個特徵

1. 出於恐懼：患得患失，無法引發積極愛心

2. 崇拜特定教主：無法轉向真正的超越界（應依法不依人）

3. 迎合世俗欲望：神明成為功能神、功利神（各廟各有「專長」）

4. 宗教排他性：敵視欺壓教外之人

九、宗教的現狀與未來

現狀問題

  • 信仰才是核心：失去信仰的大廟大教堂，只是建築
  • 世俗化威脅：終極關懷漸趨模糊
  • 新興宗教：可暫時滿足需求，能否持續取決於信仰深度

未來方向

  • 宗教不必也無法走向全球化（真正的信仰本來就是全球性的）
  • 基本教義派：應保持信仰純度，而非淪為排他工具
  • 各宗教溝通：在教義層次無法完全融合，只能相互理解尊重

十、哲學高低的判斷標準

涵蓋範圍 + 解說效率。宇宙四個層次：

| 層次 | 說明 |

|------|------|

| 物質 | 有惰性，可被控制 |

| 生命 | 自我更新 |

| 意識（直接意識） | 感知外界，動物也有 |

| 精神（反省意識） | 自我意識，唯人類有 |

唯物論涵蓋範圍最窄，解釋效率最低；能涵蓋四個層次的哲學，位階最高。

關鍵專有名詞

宗教哲學：對宗教做哲學的反省，界定信仰、超越界的本質，探討人與超越界的關係

信仰：人與超越界之間的關係，宗教的核心

超越界：既非自然界也非人類的終極存在，是信仰的對象

有位格的超越界：具有人格性（知、情、意）的神，三大一神教的超越界

非位格的超越界：境界性的存在，如涅槃、梵我一如，印度教與佛教的超越界

上帝存在論證：西方哲學史上試圖以理性論證上帝存在的嘗試，哲學的上帝未必等於聖經的上帝

輪迴：印度教與佛教的死後觀：靈魂依善惡因果不斷轉世

審判：三大一神教的死後觀：一次性，無法上訴，決定進入天堂或地獄

煉獄：天主教的死後中介狀態，供靈性淨化，最終升入天堂

四大領域：人生的四個領域：自我、群體、自然界、超越界

身心靈：人的三個層次：身體（有形）、心（知情意）、靈（與超越界相通）

直接意識：對外界的直接感知，動物也有

反省意識：意識意識到自己，即自我意識，唯人類有，是精神層次的基礎

人性向善：儒家（孔孟）的正確詮釋：人性有向善的力量，以真誠為前提，非「本善」

人性本善：宋明理學（朱熹）對儒家的誤讀，不符孔孟原義，且與基督宗教、佛教均不相容

高級宗教（湯恩比）：肯定人性不完美、反對罪惡、坦然接受苦難的宗教，引人往上超越

低級宗教：只注意人的有限需求（治病、功利），無法引人超越

迷信：出於恐懼、崇拜教主、迎合欲望、排斥異己的假宗教

依法不依人：佛教原則：以佛法（智慧道理）為依歸，而非依賴特定人物

終極關懷：保羅·田立克的概念：宗教是對人生最根本問題的關懷

Poetic justice：亞里斯多德：現實中無正義，只有文學作品中才有善惡報應

煉獄：天主教的中介狀態，供靈性淨化，最終升天堂

文化三層次：器物（身體對應）、制度（心對應）、理念（靈對應）；理念涉及終極關懷

啟蒙運動：伏爾泰語：「我不贊成你說的每一句話，但我誓死支持你說話的權利」——思想自由的根基

時間軸

約公元前551–479年：孔子生卒年，提出人性向善、仁的思想

約公元前4世紀：孟子、莊子活躍時代

1054年：天主教第一次分裂，東正教從羅馬教廷獨立

1517年：馬丁路德提出九十五題，宗教改革開始，新教（抗議教）誕生

1648年：杜斯托耶夫斯基事件原型時代（沙皇俄國政治迫害）

1844–1900年：尼采生卒年，宣告「上帝已死」

18世紀：啟蒙運動：伏爾泰、盧梭等人推動思想自由

1960年代：天主教第二次梵蒂岡大公會議，宣布不信耶穌者亦可憑良心得救

重要人物

孔子（551–479 BC）：儒家創始人，強調人性向善、真誠為核心；「不遷怒，不二過」是對顏回的最高讚美

顏回（顏淵）：孔子最優秀的學生，「不遷怒，不二過」，僅活到40歲，象徵德行的最高境界

耶穌：基督宗教核心人物，宣稱自己是神，說過「不要只看別人眼中小草，自己眼中有大樑」等教訓，強調愛與服務，本人無任何財產

湯恩比（Arnold Toynbee）：英國歷史學家，著《歷史研究》，提出高級宗教三標準（不完美人性、反對罪惡、接受苦難）

伏爾泰（Voltaire）：法國啟蒙思想家，說出「我不贊成你每句話，但誓死支持你說話的權利」——思想自由的象徵

盧梭（Rousseau）：法國啟蒙思想家，讚美原始狀態，與伏爾泰針鋒相對

杜斯托耶夫斯基（Dostoevsky）：俄國作家，26歲被捕判死刑，沙皇特赦後在集中營度過多年，體驗到「讓你活才知道你是死的」

西塞羅（Cicero）：古羅馬政治家哲學家，論老年：「好不容易老了，沒有能力就沒有欲望，多麼清爽」

方東美：作者的老師，台灣哲學家，年輕時寫「乾坤一戲場，生命一悲劇」

朱熹（1130–1200）：南宋儒者，主張人性本善，被作者認為是對孔孟的誤讀，且與其他宗教不相容

荀子：儒家學者，主張人性本惡（本能欲望），孟子後60年出現，與孟子針鋒相對

楊振寧：物理學家，諾貝爾獎得主，曾說最新科學兩千年前佛學已談到；作者指出直觀智慧與理性驗證不可同日而語
